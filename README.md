# 🌦️ Weather Forecast Application

A backend Weather Forecast Application built using Java and Spring Boot that retrieves real-time weather and forecast information from external REST APIs. The application processes JSON responses using DTOs and follows a clean layered architecture for maintainability and scalability.

## 🚀 Features

* Fetch real-time weather information
* Retrieve weather forecast data for multiple locations
* Consume external REST APIs
* Process JSON responses using DTOs
* Layered architecture (Controller, Service, DTO)
* Clean and maintainable Spring Boot project structure

## 🛠️ Tech Stack

* Java
* Spring Boot
* REST APIs
* Maven
* JSON Processing

## 📂 Project Structure

```text
src/main/java
├── controller
├── service
├── dto
└── WeatherAppApplication
```

## ⚙️ Getting Started

### Prerequisites

* Java 17 or above
* Maven
* IDE (IntelliJ IDEA, Eclipse, VS Code)

### Clone Repository

```bash
git clone https://github.com/dg2670372/Weather-App-using-Spring-Boot-and-REST-APIs-.git
```

### Run Application

```bash
mvn spring-boot:run
```

The application will start on the configured server port.
