# Weather App using Spring Boot and REST APIs

## Overview

A Weather Forecast Application built using Spring Boot that fetches real-time weather information and forecast data through external REST APIs. The application provides current weather conditions, temperature details, and forecast information for different locations.

## Features

* Real-time weather data retrieval
* Weather forecast information
* REST API integration
* Clean MVC architecture
* Spring Boot backend implementation
* JSON data processing using DTOs

## Tech Stack

* Java
* Spring Boot
* REST APIs
* Maven
* JSON
* HTML/CSS (Frontend)

## Project Structure

* Controller Layer
* Service Layer
* DTO Classes
* Configuration Files

## Getting Started

### Prerequisites

* Java 17+
* Maven
* IDE (IntelliJ IDEA / VS Code / Eclipse)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/dg2670372/Weather-App-using-Spring-Boot-and-REST-APIs-.git
```

2. Navigate to the project directory:

```bash
cd Weather-App-using-Spring-Boot-and-REST-APIs-
```

3. Run the application:

```bash
mvn spring-boot:run
```
